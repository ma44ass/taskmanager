# taskmanager
Ajouter une tâche — créer une nouvelle tâche avec un titre, une description optionnelle, et un statut (faite ou non)
Consulter toutes les tâches — récupérer la liste complète des tâches enregistrées
Consulter une tâche précise — récupérer les détails d'une tâche à partir de son identifiant
Modifier une tâche — mettre à jour le titre, la description, ou marquer la tâche comme terminée
Supprimer une tâche — effacer définitivement une tâche de la base de données
